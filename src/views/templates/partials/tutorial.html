📖 使用教程 必读
快速使用
  1. 第一步：选代理模式 - 在「设置」页选 DevinLocal（Devin Desktop 用）或 Cascade（Windsurf IDE 用）。切换后自动重启代理生效，两种模式都支持子 Agent。
  2. 第二步：选账号模式 - 首次启动强制选 Free 或 Pro，决定注入哪个 SWE 版本：Free 注入 swe-1-6 系列，Pro 注入 swe-1-7 系列；没注入的版本走官方。
  3. 第三步：填配置 - 在「模型选择」页点 + 节点添加节点，填 Base URL 和 API Key；选中节点后点「拉取」加载模型列表，选要用的模型；可选手动设置 Protocol 下拉（anthropic / openai / gemini）；Claude/GPT 可设置思考强度；OpenAI 可开启 GPT Fast Mode。
  4. 第四步：装补丁 - 切换到「设置」页点「安装补丁」，把 Devin 内部 API 指向本代理。装完重载窗口生效。
  5. 第五步：启动代理 - 点顶部「一键启动」。运行后日志显示在设置页。
  6. 第六步：运行模式 - DevinLocal 模式在 Devin 输入框底部把模式切到 DevinLocal；Cascade 模式在 Windsurf IDE 里正常使用。
BYOK 槽位映射
  • BYOK #1 → Devin Desktop 的 Claude Opus 4 BYOK
  • BYOK #2 → Claude Opus 4 Thinking BYOK
  • BYOK #3 → Claude Sonnet 4 BYOK（可选扩展槽位）
  • BYOK #4 → Claude Sonnet 4 Thinking BYOK（可选扩展槽位）
  • 四套 API / 模型完全独立
日常使用
  • 只换 API Key 或模型：修改后会自动保存。
  • 聊天没有走代理：重新安装补丁并重载窗口。
  • 切换代理/账号模式：设置页点「切换」按钮，自动重启代理。
  • 模型列表加载失败：检查 API Key、余额、网络和日志错误。
两种代理模式（DevinLocal / Cascade）在 Mac 和 Windows 上都支持。子 Agent 并行数默认最多 6 个，可用环境变量 BYOK_SUBAGENT_MAX_CONCURRENT 调整。
